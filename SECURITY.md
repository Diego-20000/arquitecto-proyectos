# Política de Seguridad

## Reportar una Vulnerabilidad

Si descubres una vulnerabilidad de seguridad, por favor **no la reportes públicamente** en GitHub Issues.

En su lugar, envía un email a: [tu-email-seguridad@ejemplo.com]

Por favor incluye:

1. Descripción de la vulnerabilidad
2. Pasos para reproducirla
3. Posible impacto
4. Sugerencias de corrección (si las tienes)

Haremos nuestro mejor esfuerzo para responder dentro de 48 horas.

## Buenas Prácticas de Seguridad

- Nunca compartas tu `.env` en repositorios públicos
- Usa `.gitignore` para archivos sensibles
- Valida siempre las entradas del usuario
- Mantén tus dependencias actualizadas

---

Agradecemos tu responsabilidad en reportar problemas de seguridad.
