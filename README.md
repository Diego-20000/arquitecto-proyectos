# 🏗️ Arquitecto de Proyectos

Una herramienta potente para generar estructuras de proyectos a partir de archivos de texto plano.

## ✨ Características

- ✅ Sin APIs externas
- ✅ Fácil de usar
- ✅ Rápido y eficiente
- ✅ Flexible y personalizable
- ✅ Listo para VS Code y GitHub

## 🚀 Instalación Rápida

```bash
git clone https://github.com/Diego-20000/arquitecto-proyectos.git
cd arquitecto-proyectos
```

## ⚡ Uso Rápido

1. Crea un archivo de plano con la estructura de tu proyecto
2. Ejecuta: `node index.js plano.txt`
3. ¡Tu proyecto está creado!
