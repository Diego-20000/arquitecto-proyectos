Arquitecto de Proyectos

Generador inteligente de estructuras de proyectos desde archivos de texto plano

Empezar Ahora Ver en GitHub

Una herramienta poderosa y versátil para generar estructuras de proyectos profesionales de forma rápida y eficiente. Perfecta para iniciadores, desarrolladores y equipos.

Características Principales

⚡

Súper Rápido

Genera estructuras complejas en segundos. Sin esperas, sin complicaciones.

📝

Formato Simple

Archivos de texto plano con estructura clara y fácil de entender.

🎯

Flexible

Personaliza cualquier estructura según tus necesidades específicas.

🚀

Sin Dependencias

Funciona completamente offline. No requiere APIs externas.

💻

CLI Intuitivo

Interfaz de línea de comandos clara y fácil de usar.

🔒

Seguro

Código auditable y sin riesgos de seguridad.

Instalación Rápida

1. Clonar el repositorio

git clone https://github.com/Diego-20000/arquitecto-proyectos.git cd arquitecto-proyectos

2. Instalar dependencias (opcional)

npm install

3. Usar la herramienta

node index.js ejemplos/ejemplo-web.txt mi-proyecto
