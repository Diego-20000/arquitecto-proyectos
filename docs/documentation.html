Aprende a usar Arquitecto de Proyectos en profundidad

Inicio Rápido

Requisitos

  • Node.js v12 o superior
  • npm (incluido con Node.js)
  • Un editor de texto cualquiera

Instalación

git clone https://github.com/Diego-20000/arquitecto-proyectos.git cd arquitecto-proyectos npm install

Uso Básico

node index.js ejemplos/ejemplo-web.txt mi-proyecto

Esto creará una carpeta llamada mi-proyecto con la estructura del archivo de plano.

Creando tu Primer Archivo de Plano

Un archivo de plano es un archivo de texto simple que describe la estructura de tu proyecto.

Formato Básico

mi-proyecto/ ├── 📁 src/ │ ├── 📄 main.js │ └── 📄 utils.js ├── 📁 public/ │ └── 📄 index.html └── 📄 package.json

Explicación de Símbolos

  • 📁 - Indica una carpeta/directorio
  • 📄 - Indica un archivo
  • ├── - Ramificación en el árbol
  • └── - Última ramificación
  • │ - Línea vertical de continuidad

Reglas Importantes

  • La indentación determina la profundidad en el árbol
  • Usa emojis 📁 y 📄 para diferenciar tipos
  • Los nombres de carpetas deben terminar con /
  • Utiliza caracteres ASCII estándar para drawing del árbol

Opciones de Línea de Comandos

Sintaxis Básica

node index.js [archivo] [nombre-proyecto]
  • [archivo] - Ruta al archivo de plano
  • [nombre-proyecto] - Nombre de la carpeta a crear

Ejemplos

# Usar un archivo de plano node index.js ejemplos/ejemplo-web.txt mi-app # Usar archivo desde ruta absoluta node index.js C:\mis-plantillas\plantilla.txt nuevo-proyecto # Usar archivo relativo node index.js ./plantillas/api.txt backend-api

Solución de Problemas

El archivo de plano no se encuentra

Problema: Error "El archivo no existe"

Solución: Verifica que el archivo exista y que la ruta sea correcta. Usa rutas relativas desde la carpeta del proyecto.

Permisos denegados

Problema: Error "Permiso denegado"

Solución: Asegúrate de tener permisos de escritura en la carpeta destino. En Linux/Mac, prueba con sudo si es necesario.

Caracteres especiales no válidos

Problema: Error al crear la estructura

Solución: Evita caracteres especiales en nombres de carpetas/archivos. Usa solo letras, números, guiones y guiones bajos.
